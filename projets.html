Projet E4 — SISR

🏥 Déploiement et sécurisation du parc informatique INTECH Medical

Projet de déploiement massif de postes de travail Windows 10/11 intégrés au domaine Active Directory du groupe INTECH Medical. Gestion centralisée via KACE SMA, sécurisation réseau et gestion des identités.

Étapes du projet
1
Audit et inventaire du parc existant
Recensement des équipements via KACE SMA, identification des postes obsolètes.
2
Préparation des images système
Création d'images Windows 10/11 standardisées avec les logiciels métier INTECH.
3
Intégration Active Directory
Jointure domaine, création comptes AD Users, application des stratégies de groupe (GPO).
4
Livraison et documentation
Livraison aux utilisateurs finaux, formation rapide, création de la documentation technique.
Technologies utilisées
Windows 10/11 Active Directory KACE SMA Windows Server GPO DHCP DNS Halo ITSM
